# Online Retail Sales Analysis
## Project Overview
This project demonstrates an end-to-end retail analytics workflow, from raw data preparation to executive-level visualisation. 

The analysis explores revenue performance across countries and over time, using structured querying and interactive dashboards to support business decision-making. 

## Tools Used
- Excel - data validation, cleaning and exploratory analysis
- MySQL - structured querying and analytical views
- Tableau Public - interactive dashboard and KPI visualisation

## Key Insights
- Total revenue of $5.05M was generated across six countries between 2019 and 2020, indicating a stable international sales base.
- Revenue is evenly distributed geographically, with the Netherlands and Spain being the highest-performing markets.
- Monthly revenue trends show consistent performance across both years, suggesting steady demand with minimal seasonality.
- Overall Performance indicates a mature and predictable retail operation.

## Dashboard
The interactive Tableau dashboard includes: 
- Total Revenue KPI
- Revenue by Country comparison
- Monthly Revenue Trend (2019-2020)
- Responsive filters for Country and Year
